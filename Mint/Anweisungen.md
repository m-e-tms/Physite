Die Mint Website ist unter /localhost/Mint zu erreichen.
Große Teile von ihr sind KI generiert.
Damit sie von Personen besser bearbeitet werden können, soll der Style nicht in Blöcken in die PHP Dateien geschrieben werden.
Entferne sie Style Blöcke aus den PHP Dateien index.php, Anmeldung.php und Registrierung.php und Hub_Lernseite_Mint.php 
Schreibe sie stattdessen in die korrespondierenden CSS Dateien. index.css (noch nicht vorhanden), Anmeldeseite.css, Registrierung.css und Hub.css 
Stelle sicher das die Style Attribute nicht in den PHP Dateien auftauchen. Stattdessen werden sie stattdessen aus den CSS Dateien geladen.
Verändere sonst nichts an den PHP oder CSS Dateien.
